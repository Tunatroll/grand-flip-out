RuneLite Plugin

Get the Grand Flip Out plugin for RuneLite and flip right from your game client. Fully Jagex-compliant — only uses official and community APIs.

Download Plugin (.jar) Source Code (ZIP)

Option A — External Plugin Hub (Easiest)

Step 1
Install the RuneLiteHijack plugin from the RuneLite Plugin Hub (search "Hijack" in the hub).
Step 2
Open RuneLiteHijack settings and paste this URL into "External Plugin Hubs":
https://raw.githubusercontent.com/Tunatroll/grand-flip-out/gh-pages/
Step 3
Restart RuneLite. Grand Flip Out will appear in your Plugin Hub list — just toggle it on!

Option B — Load JAR Manually

Step 1
Download the .jar file above and save it somewhere you'll remember.
Step 2
Open RuneLite → Settings (wrench icon) → RuneLite section → Enable "Allow loading of external plugins".
Step 3
Copy the .jar file into your RuneLite externalplugins folder:
%USERPROFILE%/.runelite/externalplugins/ (Windows) or ~/.runelite/externalplugins/ (Mac/Linux)
Step 4
Restart RuneLite. The "Grand Flip Out" panel will appear in your sidebar — you're ready to flip!
