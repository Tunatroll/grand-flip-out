# Grand Flip Out

A RuneLite plugin for Grand Exchange flipping with live prices, flip suggestions, profit tracking, and dump alerts.

## Features

- **Live GE Prices** — Multi-source pricing from the OSRS Wiki API, RuneLite API, and official GE endpoints
- **Flip Suggestions** — Smart item recommendations based on margin, volume, and ROI
- **Profit Tracking** — Automatic trade detection with per-item and per-session profit stats
- **Dump Detection** — Real-time alerts when items crash, with buy opportunity scoring
- **GE Slot Management** — Timer-based slot tracking with margin check detection
- **Price Overlays** — See margins and ROI directly on the GE interface
- **Multi-Account** — Track profits across up to 10 accounts
- **Portfolio Advisor** — Intelligent 8-slot allocation with diversification

## Installation

Install from the RuneLite Plugin Hub — search for "Grand Flip Out".

## Data Sources

All data comes from public OSRS Wiki API endpoints. No automation, no rule-breaking.
